# Athanor

Athanor is the Elixir-based control-plane for the Azoth reactive workflow engine.

## Overview

Athanor is responsible for:
- Workflow parsing and state tracking.
- Scheduling and dispatching jobs.
- Managing retries and visibility.
- Providing the UI and API for workflow management.

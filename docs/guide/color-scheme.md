# Color Scheme

The Azoth design system uses a carefully curated color palette inspired by alchemical concepts, reflecting our commitment to transformation and precision.

## Color Palette

| Element | Color Name | Hex Code | Purpose |
|---------|-----------|----------|---------|
| Primary Base | Obsidian Ink | `#0F172A` | Deep background; represents the "void" or the container of the workflow. |
| Secondary Base | Slate Zinc | `#334155` | UI Borders and cards; reflects the industrial/Rust influence. |
| The "Azoth" Accent | Transmutation Violet | `#8B5CF6` | Primary brand color; links to Elixir and the "universal solvent" concept. |
| The "Athanor" Heat | Furnace Amber | `#F59E0B` | Action buttons/Warnings; represents the heat of the control-plane furnace. |
| The "Quicksilver" | Mercury Chrome | `#E2E8F0` | Text/Primary Icons; represents the fluid, metallic data plane. |
| Success/Flow | Reactive Teal | `#14B8A6` | Status indicators; represents the successful flow of data through channels. |

## Usage Guidelines

### Obsidian Ink (#0F172A)
Used as the primary background color for all pages and components. Represents the void or container of the workflow. This deep, almost-black shade provides excellent contrast for readable text and interactive elements.

### Slate Zinc (#334155)
Applied to UI elements such as borders, cards, and secondary backgrounds. Its industrial tone complements the technical nature of the platform while maintaining visual hierarchy.

### Transmutation Violet (#8B5CF6)
The primary brand accent color, used for links, primary buttons, and key visual elements. This vibrant purple symbolizes transformation and connects to the Elixir/alchemical theme.

### Furnace Amber (#F59E0B)
Reserved for action-oriented elements like primary buttons, warnings, and alerts. Represents the heat and energy of the control-plane furnace.

### Mercury Chrome (#E2E8F0)
The default text color and primary icon color. Its light, metallic appearance represents the fluid nature of the data plane.

### Reactive Teal (#14B8A6)
Used exclusively for success states, status indicators, and representing successful data flow through channels.

## Implementation

These colors are implemented as CSS custom properties throughout the theme, allowing for consistent application and easy maintenance.
